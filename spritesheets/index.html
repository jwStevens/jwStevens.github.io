Creating a Spritesheet (Draft)

The Goal

The goal of this tutorial is to teach you how to make animations.

What is a Spritesheet?

Below is a sprite sheet for a character called an “Ember Demon”

Now I know that this may look like 4 demons next to each other but it is intended to be one character.

I’ve labeled some areas to take note of.

The first would be the blue numbers. This is the frame number at each frame.

Second is the Green X’s and the Pink Arrows; reading this like a comic strip, you can follow the colors upwards to see each piece of flame as it rises. This is similar to real life fire as seen below.

What is really importtant to notice about this comparison is that we keep the motion going upwards, this LINE OF MOTION is important to visualize if you’re going to be animating well.

You should choose a line of motion that accents the action going on.
