Game Development and Design Guidebook

Introduction

This is a guidebook on how to make video games. The main idea behind this guidebook is to provide a framework for creating interactive ideas to those either completely new to game development or people lookgin to refine their development process. In this guidebook we use Godot, GIMP and LMMS primarily, but you could easily follow along with replacements that cover the same topic.

All of the software we use in this guidebook is open source and free of charge.

Where to go from here?

If you are not yet familiar with all software we’ve mentioned, I advise you start at The Computer The best place to start would be the page: Capturing an Idea.

What is Godot?

Game Engine

Godot is the piece of software that will allow us to strap together any of our created assets, such as music, animations, sound effects etc.

What is GIMP

2D Drawing Software

GIMP is a piece of software that allows us to layer drawings and arrange pixels to create drawings of characters, foliage and special effects.

What is LMMS

(Linux Multi-Media Studio) : Music + SFX Software

LMMS lets us arrange music and generate multi-layered sound effects that can be automatically imported by Godot.
