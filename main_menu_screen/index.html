Creating a Main Menu Screen (Draft)

Overview

We are going to look into the code of the Dreamlight menu screen and I will show you how to:

  1. Make a title graphic
  2. Start a GUI Theme, using buttons
  3. Change the current scene to another.
  4. Move the camera’s target

Requirements

This is probably the first time you’re going to be exposed to extra downloads, but they will be denoted as such : git clone https://github.com/jedStevens/<repository>

You can copy and paste the command above into your terminal to downlad the repository.

For example in this lesson we will need a Camera, and I’ve linked you to my SuperCamera2D, which contains fun features like camera shake, fade transitions and automatic targeting.

git clone https://github.com/jedStevens/supercamera2d

Go ahead and download this repository by copying and pasting the command above and running it in a terminal.
